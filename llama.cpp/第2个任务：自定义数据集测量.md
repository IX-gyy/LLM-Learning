# 使用自定义数据集进行benchmark测试
## 数据集格式
自定义图像数据集格式如下：
```
export_bench_datasets
|_
  images
  |_
    图片文件
  vllm_custom_mm_1024.jsonl
```
但是注意到jsonl文件的内容形如：`{"prompt": "Please segment the target objects in the image.", "image_files": ["images/P0126_6000_6800_10200_11000.jpg"]}`  
这是不能直接在vllm中使用的，其中`image_files`字段需要转换为**绝对地址**  
我们使用nano在文件夹下创建一个python文件，并使用python 1.py将jsonl文件转换
```python
import json
import os

# 输入输出路径
input_path = "/workspace/datasets/export_bench_datasets/vllm_custom_mm_1024.jsonl"
output_path = "/workspace/datasets/export_bench_datasets/vllm_custom_mm_1024_abs.jsonl"

# 基础路径（用于构建绝对路径）
base_path = "/workspace/datasets/export_bench_datasets"

converted_data = []

with open(input_path, 'r', encoding='utf-8') as f:
    for line in f:
        data = json.loads(line.strip())
        
        # 构建图像的绝对路径
        image_files = data.get("image_files", [])
        abs_image_paths = [
            os.path.join(base_path, img_path) if not os.path.isabs(img_path) else img_path
            for img_path in image_files
        ]
        
        # 转换为绝对路径（保留原始格式）
        converted_item = {
            "prompt": data.get("prompt", "Please segment the target objects in the image."),
            "image_files": abs_image_paths
        }
        
        converted_data.append(converted_item)

# 写入新的 JSONL 文件
with open(output_path, 'w', encoding='utf-8') as f:
    for item in converted_data:
        f.write(json.dumps(item, ensure_ascii=False) + '\n')

print(f"转换完成！共处理 {len(converted_data)} 条数据")
print(f"输出文件: {output_path}")
print(f"示例数据:")
print(json.dumps(converted_data[0], indent=2, ensure_ascii=False))
```

## 文件迁移
首先将数据集迁移到板子上，直接创建一个文件夹，在VSCode上移动到里面就可以复制了  
然后将文件从板子迁移到docker容器内部，参考命令：
```
# 从主机复制到容器
docker cp /本地路径/文件夹名 容器名或ID:/目标路径/
docker cp /本地路径/文件名 容器名或ID:/目标路径/

# 示例
docker cp ./images vllm_deploy:/workspace/datasets/
docker cp ./vllm_custom_mm_1024_abs.jsonl vllm_deploy:/workspace/datasets/
```

## 开始测量
依旧启动docker，使用下面的命令启动服务端：
```
vllm serve /workspace/models/Qwen3-VL-2B-Instruct \
  --max-model-len 8192 \
  --gpu-memory-utilization 0.75 \
  --port 8000 \
  --trust-remote-code \
  --allowed-local-media-path /workspace/datasets/export_bench_datasets \
  --limit-mm-per-prompt '{"image": 1}'
```
经过测试，这个`gpu-memory-utilization`最小可以缩到0.55  
然后启动测试端
```
vllm bench serve \
  --backend openai-chat \
  --model /workspace/models/Qwen3-VL-2B-Instruct \
  --host 127.0.0.1 \
  --port 8000 \
  --endpoint /v1/chat/completions \
  --dataset-name custom_mm \
  --dataset-path /workspace/datasets/export_bench_datasets/vllm_custom_mm_1024_abs.jsonl \
  --num-prompts 64 \
  --request-rate inf \
  --max-concurrency 1
```
