## 二、llama.cpp部署
在VSCode上用SSH连接后，就和我在本地Linux系统上部署一样  
依旧AI大战项目，*以下内容待验证*：  
```
# 确认 ARM 架构
uname -m
# 输出：aarch64

# 检查 JetPack/CUDA 版本
jetson_release -v
# 或
cat /etc/nv_tegra_release

# 查看 GPU 状态（Jetson 专用）
sudo tegrastats
```
直接克隆吧，如果缺什么依赖报错了再说:(  
不过这里说的等待30~40分钟是认真的吗 :O  
```
sudo apt update
sudo apt install -y \
  build-essential \
  cmake \
  git \
  python3 \
  python3-pip \
  gdb
git clone https://github.com/ggml-org/llama.cpp.git
cd llama.cpp

mkdir build
cd build
cmake ..
cmake --build . -j
```
搞一个小一点的模型试一下：
```
# 回到项目根目录
cd ..

# 创建模型目录
mkdir -p models

# 下载测试模型（TinyLlama，1.1GB，速度快）
wget https://huggingface.co/TheBloke/TinyLlama-1.1B-Chat-v1.0-GGUF/resolve/main/tinyllama-1.1b-chat-v1.0.Q4_K_M.gguf -P models/

# 简单推理
./build/bin/llama-cli \
  -m models/tinyllama-1.1b-chat-v1.0.Q4_K_M.gguf \
  -p "你好，请用中文介绍自己" \
  -n 256 \
  -e
```
